# OuiRise // Scale Your Understanding

**Protocol:** 6.7 (Vertical Slicing)  
**Stack:** Next.js + React + TypeScript + Tailwind  
**Database:** MongoDB  
**Theme:** Dynasty Dark

---

## The Love Language (Embedded in Architecture)

- **Fast load** (<100kb) - respects your time
- **No surveillance** - no Google, no Facebook, no tracking pixels
- **Clear exit** - opt-in data, no lock-in
- **Semantic safety** - no "users", no "leads", no extraction

---

## Quick Start

```bash
# Install dependencies
npm install

# Set up environment
cp .env.local.example .env.local
# Add your MongoDB URI to .env.local

# Run development server
npm run dev

# Build for production
npm run build
```
